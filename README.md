# e-Konfident - Obywatelskie Zgłoszenia

![Aplikacja e-Konfident](https://i.ytimg.com/vi/PCThjFBU1P0/maxresdefault.jpg)

## O aplikacji
e-Konfident to mobilna aplikacja PWA (Progressive Web App), która umożliwia obywatelom szybkie dokumentowanie i zgłaszanie wykroczeń (np. źle zaparkowanych pojazdów, wandalizmu, dzikich wysypisk śmieci). 

Aplikacja wykorzystuje fizyczne funkcje urządzenia:
1. **Kamerę** - do zrobienia zdjęcia wykroczenia.
2. **GPS** - do pobrania dokładnej lokalizacji zdarzenia.
3. **Web Share API** - do łatwego udostępnienia zgłoszenia (np. na policję, straż miejską lub do znajomych) wraz z mapą.
